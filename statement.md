# Secure Personal Finance Tracker - Project Statement

## 1. Problem Statement
The current difficulty individuals face in manually tracking personal income and expenses across various sources, leading to fragmented financial records and poor overall financial visibility.

## 2. Scope of the Project
This project is a **console-based Java application** focused on the core functionality of personal finance management. It is scoped to handle **user authentication**, **CRUD operations** on transactions, and simple **summary reporting** with data export capability. It utilizes **JDBC/JPA** for persistent storage.

## 3. Target Users
* **Students** and **Individuals** who need a straightforward, secure tool to log and categorize daily income and expenses.
* Users who prefer a **command-line interface** for quick data entry and reporting.

## 4. High-Level Features
* User registration and secure login.
* Ability to add, view, edit, and delete income and expense transactions.
* Categorization of all transactions.
* Generation of monthly financial summaries.
